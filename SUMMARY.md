## Real Example Program

```hausa
# Learn programming in Hausa
aiki fibonacci(n):
    idan n <= 1:
        mayar n
    mayar fibonacci(n - 1) + fibonacci(n - 2)

suna = "Ali"
rubuta "Sannu " + suna

adadi = 7
result = fibonacci(adadi)
rubuta "Fibonacci of " + adadi + " = " + result
```
